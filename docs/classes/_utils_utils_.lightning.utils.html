Static uniqueRange

  • uniqueRange(from: number, to: number, count: number, safeNumber?: number): number[]
    • Defined in utils/utils.ts:73
    description

    Return a unique array of number between two points. Pass an optional safeNumber which cuts the while loop out if you enter an infinite loop. This is set to 5 times your total number, which I do not recommend you remove!

    example
    
Lightning.Maths.uniqueRange(0, 10, 5) // will return an array of 5 unique numbers between 0 and 10

    Parameters

    • from: number
    • to: number
    • count: number

      Number of unique values

    • Default value safeNumber: number = count * 5

    Returns number[]
