game: Engine
  • Defined in controllers/scale/scale.ts:26

Position Horizontally Position Vertically New positions & aspect ratios

Scale Config - : 0 - No scaling - Done 1 - Maximum width (keeping AR) - Done 2 - Maximum height (keeping AR) - Done 3 - Maximum width and height (keeping AR) - Done 4 - Stretch to fix maximum width - Done 5 - Stretch to fit maximum height - Done 6 - Stretch to fit full width and full height - Done 7 - Stretch Full Screen 8 - Scale Full Screen
